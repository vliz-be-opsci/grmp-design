# grmp-design

This repository contains the analysis (functional and technical) for the GRMP project.

This analysis can be found in different markdown documents.

We make use of a basic git pull request process to make sure that propositions and changes are
approved.

### Repositories
* [GRMP design & analysis](https://github.com/vliz-be-opsci/grmp-design)
* [GRMP orchestrator framework](https://github.com/vliz-be-opsci/grmp-framework)
* [GRMP tests](https://github.com/vliz-be-opsci/grmp-test-implementations)
* [GRMP demo](https://github.com/vliz-be-opsci/grmp-demo): a demo repository that can be used as a 
  starting point to create and configure your own tests suite
